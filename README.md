## 📋 О проекте

**SkillTree** — это платформа для онлайн-обучения. Цель — создать удобное web-приложение для размещения курсов, заданий и отслеживания прогресса студентов. Каждый желающий может зайти и создать свой курс. 

Проект находится в активной разработке.

---

## 🛠 Технологический стек

| Компонент | Технологии |
|-----------|------------|
| **Язык** | Java 21 |
| **Фреймворк** | Spring Boot 3.x, Spring Data JPA / Hibernate |
| **База данных** | PostgreSQL 16 |
| **Миграции** | Flyway |
| **Сборка** | Maven |
| **Контейнеризация** | Docker, Docker Compose |
| **CI/CD** | GitHub Actions |
| **Тестирование** | JUnit, Mockito |

## 🚀 Запуск

Terminal 1:
```
cd backend/ && docker-compose up --build
```

Terminal 2:
```
cd frontend && npm run dev
```
